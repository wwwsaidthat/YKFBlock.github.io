## C 与 C++ 的关系

1.  C++是C的超集  
2. C++支持 C 所支持的全部编程技巧  
3. 任何 C 程序都能被 C++ 用基本相同的方法编写，并具备同等开销（时间、空间）







## 目的

更高效地进行编码

## 设计理念

1. 效率
2. 实用性优于艺术性严谨性
3. 允许一个有用的特征比防止各种错误使用更重要（相信程序员）
