   int left = 0 , right = size - 1;

2. 左闭右闭

```c++
while(left <= right){
    mid = (left + right ) / 2;
    if(arr[mid] < target){
        left = mid + 1;
    }else{
        right = mid - 1;
    }
}
return left;
```
