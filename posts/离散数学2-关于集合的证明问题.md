# 集合

1. 集合元素的性质

   1. 确定性
   2. 唯一性
   3. 无顺序性

2. 重点掌握的运算

   1. 差集 定义
      $$
      A - B  \iff A \cap \overline{B}
      $$
      

   2. 对称差

      对称差定义
      $$
      A \oplus B = (A - B) \cup (B - A)
      $$
      对称差**对称性**
      $$
      A \oplus B \iff B \oplus A
      $$
      对称差**结合性**
      $$
      A \oplus (B \oplus C) = (A \oplus B) \oplus C
      $$
      

   3. 幂集 -- 重点理解{x} 和 x
      $$
      A = X \cup \{X\}
      \\ X \in A
      \\ X \subseteq A
      \\ 幂集定义： A = \{x \subseteq X \}
      $$
      

3. 证明思路总结

   1. 证明集合相等：互为子集（最常用的，这个不行在用别的）
   2. 思想：集合是由元素确定的
   3. 反证法
   4. 文氏图证明法--要求进行适当步骤的展开
   5. 具体问题具体分析 有一些巧妙的方法但是不是通法，结合具体的性质临场发挥
   6. 很多证明题依靠定义就能解决（例如对称差的定义往往是解决对称差证明题的关键）
   7. 然后就是性质（例如差集的性质）

4. 例题1. 证明空集只有一个
   $$
   假设有两个空集\phi_1 \,\, \phi_2\\
   \phi_1 \subseteq \phi_2\\
   \phi_2 \subseteq \phi_1\\
   \phi_1 = \phi_2\\
   $$
   

5. 例题1. （利用2）$A \subseteq X$ $B\subseteq X$ 求证$A \cup B \subseteq X$
   $$
   x \in A \cup B \iff x \in A \,\, or \,\, B\\
   x \in A  \,\,\,  A \subseteq X \,\,\, x \in X\\
   x \in B  \,\,\,  B \subseteq X \,\,\, x \in X\\
   \because x的任意性
   \therefore A \cup B 所有的元素都是X的元素，即A \cup B \subseteq X
   $$
   

6. 例题2. （反证法）$A \oplus B = A \oplus C$求证 B = C
   $$
   B \neq C\\
   \exist x \in B ,x\notin C\\
   x \in A,x\notin A \oplus B,x\in A \oplus C,与条件矛盾\\
   x \notin A, x\in A \oplus B,x\notin A \oplus C，与条件矛盾\\
   得证\square
   $$
   

7. 例题3. （用文氏图证明）$A \oplus (B \oplus C) = (A \oplus B) \oplus C$ 要进行展开化简 方法就是把括号外面的对称差按照定义展开，然后再画出来才行，**老师强调必须要写出怎么画出来的，不能直接画**

8. 例题4. （巧妙利用性质证明）$A \oplus B = A \oplus C$求证 B = C
   $$
   左右两边同时在左边加A\oplus\\
   A\oplus A \oplus B = A\oplus A \oplus C\\
   得到B = C
   $$
   利用的性质：

   1. $A \oplus A = \phi$
   2. $A \oplus \phi = A$
   3. $A \oplus (B \oplus C) = (A \oplus B) \oplus C$  这个性质是用来告诉你加上之后先算A和B和先算A和A是一样的，这样变化不会改变运算结果
