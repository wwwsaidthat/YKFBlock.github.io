1. 简叙Goedel第二不完全定理及其证明思路。

    对于**足够强**且**和谐**的可递归公理化形式系统 S，首先借助**不动点引理**构造哥德尔句 G，使其满足 S⊢G↔¬ProvS(┌G┐)；再通过**反射**原理刻画系统的可证性内省性质，建立 S⊢Con(S)→G 的蕴含关系；反证假设 S⊢Con(S)，则由分离规则推出 S⊢G，这与第一定理中 “**和谐**的 S 内 G 不可证” 的结论矛盾，因此 Con(S) 无法在 S 内证明。

2. 定理30A的证明思路。

    1. 造自指句：定义可定义的三元关系R，构造语句$\sigma$，其含义是“没有从A出发的推理能推出我”。
    2. 反证出矛盾：假设A能推出$\sigma$，会导出$\sigma$在$\mathfrak{N}$中既真又假的矛盾，故A推不出$\sigma$。
    3. 证$\sigma$为真：由A推不出$\sigma$，结合R的定义与替换引理，可证$\sigma$在$\mathfrak{N}$中为真。

    最终结论：存在$\mathfrak{N}$中为真但A推不出的句子，$\mathrm{Th}\mathfrak{N}$不可公理化。

4. AE公理集有哪些？分别是什么？
    AE公理集通常指Robinson算术公理（System Q），共有9条：
    1. $\forall x (Sx \neq 0)$  (0不是任何数的后继)
    2. $\forall x \forall y (Sx = Sy \to x = y)$ (后继函数是单射)
    3. $\forall x (x \neq 0 \to \exists y (x = Sy))$ (除了0以外的数都是后继)
    4. $\forall x (x + 0 = x)$ (加法定义1)
    5. $\forall x \forall y (x + Sy = S(x + y))$ (加法定义2)
    6. $\forall x (x \cdot 0 = 0)$ (乘法定义1)
    7. $\forall x \forall y (x \cdot Sy = (x \cdot y) + x)$ (乘法定义2)
    8. $\forall x \forall y (x < y \leftrightarrow \exists z (z \neq 0 \land x + z = y))$ (小于号定义)
    9. $\forall x \forall y (x < y \lor x = y \lor y < x)$ (全序关系)

5. 可表示、可定义是怎么定义的？他们有什么区别？

    可定义：<a~1~,...a~m~> $\in$ R $\iff$ $\models\rho(S^{a1}0,...S^{am}0)$

    可表示：一个关系可表示当且仅当存在一个公式，它从T中表示该关系

    对于可定义性，我们关心的是句子的解释是否是真的；而对于在C~n~ A~E~ 中可表示，我们关心的句子是否能从公理中推出

7. 不动点引理的证明思路？

     对**足够强**的形式系统 S 与任意一元谓词 ψ(x)，先通过**哥德尔编码**将公式映射为自然数，定义递归的**代入函数** sub(x,y)（把哥德尔数为 x 的公式中自由变元代换为 y 并输出新公式的哥德尔数），再构造对角线公式 α(x,y)=ψ(Sub(x,x,y))，取其哥德尔数 n 并代入得到闭公式 φ=α(n,n)，最后利用系统对递归函数的可表示性，验证得出 S⊢φ↔ψ(┌φ┐)，即证得满足自指等价关系的不动点公式 φ 存在。
