### 1) 单例模式（Singleton）

题目：实现一个全局唯一的 `ConfigCenter`，要求：
- 线程安全
- 延迟加载（第一次用到再创建）
- 外部不能 `new`

参考要点：
- 构造方法私有化

- 对外提供 `getInstance()`

- 推荐写法：静态内部类（Initialization-on-demand holder）

  ```java
  class ConfigCenter{
    private static ConfigCenter instance;
    private ConfigCenter(){}
    public static ConfigCenter getInstance(){
      if(instance == null) instance = new ConfigCenter();
      return instance;
    }
  }
  ```

### 5) 建造者模式（Builder）

题目：实现一个 `HttpRequest` 构建过程，字段多且可选：`method/url/headers/query/body/timeout`。要求：
- 可读性强（链式调用）
- 构建后对象不可变

参考要点：
- `HttpRequest.Builder` 持有可变字段，`build()` 产出不可变 `HttpRequest`
- 适合“参数多、组合复杂、构建步骤固定但细节可变”的场景

```java
abstract class Builder{
  public abstract Builder buildPart(String part);
  public abstract HttpRequest getResult();
}
class HttpRequestBuilder{
  private HttpRequest req = new HttpRequest();
	public HttpRequestBuilder buildPart(String part){
    if (part == null || part.isEmpty()) return this;

     if (part.startsWith("method=")) {
         req.setMethod(part.substring("method=".length()));
     } else if (part.startsWith("url=")) {
         req.setUrl(part.substring("url=".length()));
     } else if (part.startsWith("timeoutMs=")) {
         req.setTimeoutMs(Integer.parseInt(part.substring("timeoutMs=".length())));
     } else if (part.startsWith("body=")) {
         req.setBody(part.substring("body=".length()));
     } else if (part.startsWith("header:")) {
         String kv = part.substring("header:".length());
         int idx = kv.indexOf('=');
         if (idx > 0) req.putHeader(kv.substring(0, idx), kv.substring(idx + 1));
      }
        return this;
  }
  public HttpRequest getResult(){
    return req;
  }
 
}
class HttpRequest{
  private String method;
  private String url;
  private Map<String,String> headers;
  private String query;
  private String body;
  private int timeout;
  
  public void setMethod(String method){
    this.method = method;
  }
  public void setUrl(String url){
    this.url = url;
  }
  public void putHeader(String k,String v){
   	this.headers.put(k,v);
  }
  public void setQuery(String query){
    this.query = query;
  }
  public void setBody(String body){
    this.body = body;
  }
  public void setTimeout(int timeout){
    this.timeout = timeout;
  }
  @Override
    public String toString() {
        return "HttpRequest{" +
                "method='" + method + '\'' +
                ", url='" + url + '\'' +
                ", headers=" + headers +
                ", body='" + body + '\'' +
                ", timeoutMs=" + timeoutMs +
                '}';
    }
}
public class Main {
    public static void main(String[] args) {
        HttpRequest req = new HttpRequestBuilder()
                .buildPart("method=POST")
                .buildPart("url=https://example.com/api")
                .buildPart("header:Content-Type=application/json")
                .buildPart("timeoutMs=3000")
                .buildPart("body={\"a\":1}")
                .getResult();

        System.out.println(req);
    }
}
```

### 7) 组合模式（Composite）

题目：实现文件系统：`File` 和 `Folder` 都要支持 `getSize()`；`Folder` 内部可以包含 `File` 或 `Folder`，并递归统计大小。

参考要点：
- 抽象组件 `Node`：统一对待叶子和容器
- 容器节点维护子节点集合

```java
import java.util.List;
import java.util.ArrayList;

abstract class Node{
  protected String name;
  protected int size;
  public abstract int getSize();
  public abstract void show();
  public abstract void Add(Node node);
  public abstract void Remove(Node node);
  public abstract List<Node> getChildren();
}
class File extends Node{
  public File(String name,int size){
    this.name = name;
    this.size = size;
  }
  @Override
  public int getSize(){
    return size;
  }
  @Override
  public void show(){
    System.out.println("The size of "+name + " is " +size);
  }
  @Override
  public void Add(Node node){
    return;
  }
  @Override
  public void Remove(Node node){
    return;
  }
  @Override
  public List<Node> getChildren(){
    return null;
  }
}
class Folder extends Node{
  public Folder(String name){
    this.name = name;
    this.size = 0;
  }
  private List<Node> files = new ArrayList<>();
  public List<Node> getChildren(){
    return files;
  }
  @Override
  public int getSize(){
    size = 0;
   	for(int i = 0 ; i < files.size(); i++){
      Node file = files.get(i);
      size += file.getSize();
    }
    return size;
  }
  @Override
  public void show(){
    System.out.println("The size of "+name + " is "+ getSize());
  }
  public void Add(Node node){
    files.add(node);
  }
  public void Remove(Node node){
    files.remove(node);
  }
}


public class Main{
  public static void main(String[] args){
    Node root = new Folder("root");
    Node file1 = new File("wwwsaidthat.md",10);
    Node file2 = new File("happy.png",20);
    Node folder1 = new Folder("root2");
    Node file3 = new File("happy2.png",30);
    Node file4 = new File("sad.png",20);
    root.Add(file1);
    root.Add(file2);
    root.Add(folder1);
    folder1.Add(file3);
    folder1.Add(file4);
    
    root.show();
  }
}
```

### 8) 桥接模式（Bridge）

题目：画图系统：形状 `Circle/Rect` 与颜色 `Red/Blue` 可以自由组合，如果用继承会产生类爆炸（RedCircle、BlueCircle...）。用桥接消除组合爆炸。

参考要点：
- 抽象维度（Shape）持有实现维度（Color）的引用
- `Shape.draw()` 委托给 `Color.applyColor()`

```java
abstract class Shape{
  protected Color color;
  public abstract void draw();
}
class Circle extends Shape{
  private int r;
  public Circle(int r){this.r = r;}
   public void draw(){
     System.out.print("draw a circle with a radius = "+r +" with color : ");
     color.applyColor();
     
   }
}
class Rect extends Shape{
  private int a;
  private int b;
  public Rect(int a,int b){this.a = a ; this.b = b;}
  public void draw(){
    System.out.print("draw a rectangular with a = "+a+" b = "+b + " with color : " );
     color.applyColor();
  }
}
interface Color{
  void applyColor();
}
class Red implements Color{
  public void applyColor(){
    System.out.println("Red");
  }
}
class Blue implements Color{
  public void applyColor(){
    System.out.println("Blue");
  }
}
public class Main{
  public static void main(String[] args){
    Circle circle = new Circle(5);
    circle.color = new Red();
    circle.draw();
    Rect rect = new Rect(3,4);
    rect.color = new Blue();
    rect.draw();
  }
}
```
