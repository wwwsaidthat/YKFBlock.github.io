# 装饰模式

子类继承父类，子类拥有父类的成员对象，双重关系，存在**伦理问题**！！！

父亲生我，我中有父亲！

transform有个儿子叫做changer

changer类中有个成员变量是transform用来初始化

```java
class Transform{
  void move(){
    ...
  }
}
class Car extends Transform{
  void move(){
    ...
  }
}
class Changer extends Transform{
  protected Transform transform;
  public Changer(Transform transform){
    this.transform = transform;
  }
  public void move(){
    ...
  }
}

class Robot extends Changer{
  public Robot(Transform transform){
    this.transform = transform;
  }
}

class Plane extends Transform{
  public Plane(Transform transform){
    this.transform = transform;
  }
}
```
