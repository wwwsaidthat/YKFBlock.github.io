小蓝有一个序列 a*a*，同时他的朋友小乔有一个序列 b*b*。

我们认为两个序列是和谐的，当且仅当存在某个数 x*x*，使得对于所有的 i*i* 有 ai+x=bi*a**i*+*x*=*b**i*。

现在小蓝可以让序列 a*a* 旋转，即通过一次参数为 k*k* 的旋转可以将序列 a1,a2,⋯,an *a*1,*a*2,⋯,*a**n* 变为 a1+k,a2+k,⋯,an,a1,a2,⋯,ak*a*1+*k*,*a*2+*k*,⋯,*a**n*,*a*1,*a*2,⋯,*a**k*。

小蓝希望知道，是否存在这样的旋转操作，能够让序列 a*a* 和 b*b* 是和谐的。

输出共一行，一个自然数 k*k* 表示参数为 k*k* 的旋转操作能够让 a,b*a*,*b* 是和谐的，如果存在多个这样的 k*k*，请输出最小的 k*k*，如果不存在这样的 k*k*，请输出 −1−1。



使用双向队列dq 注意引入头文件

```c++
#include <iostream>
#include <deque>
#include "vector"
using namespace std;
int main()
{
    // 请在此输入您的代码
    int n;
    cin >> n;

    vector<int> a(n),b(n);
    for(int i = 0 ; i < n ; i++) cin >> a[i];
    for(int i = 0 ; i < n ; i++) cin >> b[i];
    deque<int> dqa,dqb;
    dqa.push_back(a[0] - a[n - 1]);
    dqb.push_back(b[0] - b[n - 1]);
    for(int i = 1 ; i < n ; i++){
        dqa.push_back(a[i] - a[i - 1]);
        dqb.push_back(b[i] - b[i - 1]);
    }
    int tmp = 0;
    for(int i = 1 ; i < n ; i++){
        tmp = dqa.front();
        dqa.pop_front();
        dqa.push_back(tmp);
        if(dqa == dqb){
            cout << i;
            return 0;
        }
    }
    cout << -1;
    return 0;
}
```
