### 问题描述

小蓝是蓝桥王国的卫队长，他手下有 n*n* 名战士，第 i*i* 名战士的战斗力为 ai*a**i*。他可以任选若干名战士组成一个 **非空的** 战斗小队，如果战斗小队中战斗力最弱加上战斗力最强不大于指标 k*k*，则该选法被认定为完美选法。现在小蓝想知道他能选出多少种不同的完美小队。答案可能很大，你需要对 109+7109+7 取模。

### 输入格式

第一行输入两个整数 n*n* 和 k*k* ，表示战士的人数和完美选法的指标。

第二行输入 n*n* 个整数，第 i*i* 个数表示第 i*i* 名战士的战斗力。

数据范围保证：1≤n≤1051≤*n*≤105，1≤ai,k≤1061≤*a**i*,*k*≤106。

### 输出格式

输出一个整数表示答案，答案需要对 109+7109+7 取模。

### 样例输入

```text
3 3
1 2 3
```

### 样例输出

```text
2
```

### 说明

有以下两种选法符合要求：

[1][1] ：最小值加最大值为 (1+1)<3(1+1)<3 。

[1,2][1,2]：最小值加最大值为 (1+2)≤3(1+2)≤3。

所以样例答案为 22。

```c++
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const ll num = 1e5 + 5;
ll n,a[num],k,sum;
const ll mod = 1e9 + 7;
ll pow2(ll x){
  ll ret = 1,t = 2;
  while(x){
    if(x & 1 ) ret = ret * t % mod;
    t = t * t % mod;
    x = x >> 1;
  }
  return ret;
}
int main(){
  cin >> n >> k;
  for(int i = 0 ; i < n ; i ++)cin >> a[i];
  sort(a,a+n);
  for(int i = 0 ; i < n; i++){
    //min = a[i] max = k - a[i] 2 ^ (maxPos - minPos)
    if(a[i] + a[i] > k) break;
    ll target = k - a[i];
    auto it = upper_bound(a,a+n,target);
    ll maxPos = it - a - 1;
    sum = (sum + pow2(maxPos - i)) % mod;
  }
  cout << sum % mod ;
  return 0;
}
```
