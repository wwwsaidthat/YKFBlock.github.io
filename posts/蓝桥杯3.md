蓝桥学院的冬季旅行队终于到达了冰雪奇缘大世界。只见门口大招牌上挂着一个长度为 N*N* 的小写字符串 S*S*，经过了解，原来冰雪奇缘大世界每日都会推出一个回文字符串挑战，挑战成功即可获得一份精美礼品。挑战规则如下：

- 可以任意重排字符串 S*S* 的字符顺序。
- 重排后，需要将整个字符串切割为若干个连续的回文子串（每个子串必须是回文）。
- 求出切割后最少的回文子串数量。

现在请你帮助旅行队解决这个问题，输出这个最小数量。



注意只有一个字符串的情况不要输出0

```c++
#include <iostream>
#include <map>
using namespace std;
int main()
{
  // 请在此输入您的代码
  int N;
  cin>> N;
  string S;
  cin>>S;
  map<char,int> m;
  int ret = 0;
  for(int i = 0 ; i < N;i++){
    m[S[i]]++;
  }
  for(auto&[k,v] : m){
    if(v % 2 == 1) ret++;
  }
  if(ret == 0) ret ++;
  cout << ret;
  return 0;
}
```
