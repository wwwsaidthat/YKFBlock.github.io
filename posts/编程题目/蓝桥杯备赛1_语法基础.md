9. 二分查找

   ```c++
   int l = 0,r = n-1;
   while(l<r){
       int mid = l + r >> 1;
       if(check(mid)) r = mid;
       else l = mid + 1;
   }
   ```
   ```c++
   bool ans = binary_search(l,r,check);
   ```
   lower_bound(l,r,check); // >=
