求阶乘

### 问题描述

满足 N! 的末尾恰好有 K个 0 的最小的 N 是多少?

如果这样的 N 不存在输出 −1 。

### 输入格式

一个整数 K 。

### 输出格式

一个整数代表答案。

### 样例输入

```text
2
```

### 样例输出

```text
10
```

### 评测用例规模与约定

对于 30%的数据, 1≤K≤10 6

对于 100% 的数据, 1≤K≤10 18

```c++
#include<bits/stdc++.h>
using namespace std;
using ll = long long;
  ll K;
int check(ll m){
  ll ret = m;
  while(m) {
    ret += m / 5;
    m /= 5;
  }
  return ret >= K;
}
int check2(ll m){
  ll ret = m;
  while(m) {
    ret += m / 5;
    m /= 5;
  }
  return ret == K;
}
int main(){
  cin >> K;
  ll l = 0,r = K;
  while(r - l > 1){
    ll m = (l + r) >> 1;
    if(check(m)) r = m;
    else l = m;
  }
  ll ans = (check(l)) ? l : r;

  if(check2(ans)) cout << ans * 5;
  else cout << -1;
  return 0;
}
```
