### 问题描述

肖恩认为一般的乘法表不够美观，因为它是三角形的所以肖恩认为不够整齐。肖恩自己制作了一张矩形乘法表，对于一张 n×m的矩形乘法表，肖恩会把 i×j填充到矩形的第 i行第 j 列的位置。现在，肖恩向你提问：在这张乘法表中，第 k 大的元素是多少？

矩形乘法表中第 k 大元素是指将矩形乘法表k中所有元素从小到大排列后的第 k 个元素。

### 输入描述

输入三个数字 n,m,k ，每个数字的意义和问题描述中相同。

输入保证 1≤n,m≤5×$10^5$,1≤k≤n×m 。

### 输出描述

输出一个数字表示第 k大的元素。

### 样例输入

```
2 4 5
```

### 样例输出

```
4
```

```c++
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
ll n,m,k;
int check(ll mid){
  //i*j <= m
  ll cnt = 0;
  for(int i = 1;i <= n;i++){
    cnt += min(m,mid/i);
  }
  return cnt > k;
}
int main(){
  cin>>n>>m>>k;
  ll l = 1,r = n*m+1;
  while(r-l>1){
    ll mid = (r + l)>>1;
    if(check(mid)) r = mid;
    else l = mid;
  }
  if(check(l))cout << l;
  else cout << r;
  return 0;
}
```

关键点在于check，数m怎么计算比mid小的数有几个
对于一个数i，比m小的数有min(m,mid/i)个
按照一行来看，第i行比mid小的数有min(m,mid/i)个
首先是最多一行m个数，另外就是第i行有mid/i个数比mid小
