void dfs(Node* root){

广搜

```c++
// 四个方向：下、左、上、右
int direction[4][2] = {
    {1,0}, {0,-1}, {-1,0}, {0,1}
};

// 假设：地图是 grid，vis 是访问标记，n、m 是行列数
int n, m;
int grid[105][105];
bool vis[105][105];

void bfs(int startX, int startY) {
    // 队列存 坐标 (x,y)
    deque<pair<int, int>> dq;

    // 起点入队 + 标记访问
    dq.push_back({startX, startY});
    vis[startX][startY] = true;

    while (!dq.empty()) {
        // 取出队首
        auto now = dq.front();
        dq.pop_front();

        int x = now.first;   // 当前点 x
        int y = now.second;  // 当前点 y

        // 遍历 4 个方向
        for (int i = 0; i < 4; i++) {
            int nx = x + direction[i][0];
            int ny = y + direction[i][1];

            // ✅ 判断：不越界 + 未访问 + 可以走（根据题目改条件）
            if (nx >= 0 && nx < n && ny >=0 && ny < m && !vis[nx][ny] && grid[nx][ny] == 1) {
                vis[nx][ny] = true;          // 标记访问
                dq.push_back({nx, ny});      // 入队
            }
        }
    }
}
```

dfs最难的就是遍历之后的回溯过程
建议的方法就是把所有的变量全都检查一遍，整个思路就是目前脚下一共有多少条路，都试一下
模板就是
solve(condition1)
dfs(condition1)
recover(condition1)
solve(condition2)
dfs(condition2)
recover(condition2)
....
